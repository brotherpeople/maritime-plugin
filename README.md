A Figma plugin that generates device bezels around images, frames, and vector objects.
Adjust stroke weight and corner radius, add shadows, and overlay a steering wheel graphic — all in real-time.

---

## Installation

> **Note:** Figma desktop app is required. This plugin cannot be loaded from the browser version.

1. Go to the [Releases](../../releases) page and download the latest ZIP file.
2. Unzip the downloaded file to a folder on your computer.
3. Open the **Figma desktop app**.
4. Navigate to **Main Menu (☰) → Plugins → Development → Import plugin from manifest...**
5. Open the unzipped folder and select the `manifest.json` file.
6. The plugin will appear under **Plugins → Development** and is ready to use.
